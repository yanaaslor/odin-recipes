# odin-recipes

This project is part of the curriculum from The Odin Project, covering Git Basics 
and HTML Foundations. It involves building a simple recipe website using basic HTML.

By completing this project, I practiced and demonstrated the following skills:

    - Navigating and managing files and directories using the command line
    - Creating and managing a GitHub repository
    - Cloning repositories and working with Git
    - Using Git for version control (add, commit, push)
    - Writing basic HTML to structure web pages
